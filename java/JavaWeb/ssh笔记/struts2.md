## 概述
Struts2 是Apache开源的一个MVC框架
什么是MVC
    MVC是由Model(模型),View(视图),Controller(控制器)组成
    Model：负责数据维护
    View：负责向用户呈现数据
    Controller：控制Model与View的交互

## Struts2的架构
Struts2的MVC架构主要通过五个核心实现的
- 操作(Action)
- 拦截器(Interceptors)
- 值栈(Value Stack) / OGNL
- 结果(Result) / 结果类型
- 视图技术
